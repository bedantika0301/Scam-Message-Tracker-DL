1. Goal of the Project:
The goal of this project is to create a Scam Message Tracker. This tracker will help users check if text messages are safe or not.

2. Background:
People use SMS, emails and social media every day. This makes it easy to talk to others. Also increases the risk of online scams. Scammers send messages that look real. These messages can be about lotteries, phishing links or OTP-based frauds.

3. Problem:
Many users can't tell if a message is a scam or not. This is because scam messages look genuine and make users feel like they need to act

4. Solution Approach:
This project uses machine learning and keyword analysis to solve the problem.
* A classification model based on the Multinomial Naive Bayes algorithm is trained on sample messages.
* The input text is converted to lowercase. Cleaned before analysis to improve accuracy.
* The system then analyzes the message. It predicts if it is a scam message. It checks for keywords like "OTP," "" "win," and "free.”

5. System Features:
* The system detects if a message is a scam or safe.
* It highlights keywords that could be risky.
* It keeps a history of analyzed messages.
* It provides a summary of results.

6. Benefits:
The Scam Message Tracker helps users see patterns in messages. It helps them understand risks. The system is simple and easy to use. Even people with technical knowledge can use it effectively.

7.
The Scam Message Tracker helps detect scam messages. It raises awareness among users. It encourages people to be more careful while communicating online. This project shows how artificial intelligence can improve safety. The tracker is a tool, for staying safe in the digital world.
